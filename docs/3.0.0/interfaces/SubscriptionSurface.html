    Interface SubscriptionSurface<TEventMap>

    The public subscription and introspection surface of Bus, excluding Bus.emit. Bus.listeners and Bus.ownListeners remain on Bus only, since ReadonlyMap keys are invariant in the event map type parameter and would break contravariant views.
